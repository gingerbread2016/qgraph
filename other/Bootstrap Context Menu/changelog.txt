Twitter Bootstrap Context Menu 2012-01-12
----------------------------------------
- Added  "beforeDisplay" callback function.
    example: 

        <div style="width: 400px;
            height: 60px;
            background: #EEE;
            font-weight: bold;
            padding: 5px;
            border: 1px solid #CCC;" id="id8">Right click here</div>

        <script>
        var menu = {}; // define a x-context menu object.

        menu['copy'] = {text:'Copy'}; // add Copy element to menu.
        menu['paste'] = {text:'Paste'}; // add Paste element to menu.
        menu['paste_shortcut'] = {text:'Paste shortcut'}; // add Paste shortcut element to menu.

        $('#id8').contextMenu(menu); // register menu to a html tag

        $('#id8').contextMenu('beforeDisplay',function(target){ 
                                                console.log(target.attr('id')) 
                                            });
        </script>
